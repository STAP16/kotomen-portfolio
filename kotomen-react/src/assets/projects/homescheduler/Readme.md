Homescheduler Homescheduler — приватное веб‑приложение для управления семейными задачами и бытовыми
процессами, разработанное для реального повседневного использования.

Приложение интегрировано в Telegram Mini App, что позволяет работать с задачами прямо внутри
Telegram без отдельной установки и регистрации в сторонних сервисах.

🎯 Назначение проекта Homescheduler решает задачу централизованного управления семейными делами:

прозрачное распределение задач между участниками семьи; контроль выполнения и сроков; единое
пространство для взаимодействия без лишних инструментов. Ознакомится подробнене с концепцией
приложения: Ссылка image image image image 🧩 Ключевые возможности Управление задачами Создание,
редактирование и удаление задач Статусы выполнения (выполнена / не выполнена) Сортировка задач
Контроль просроченных задач Многопользовательская работа Поддержка нескольких участников (члены
семьи)

Ролевую модель:

Администратор — управление пользователями и задачами Участник — работа с назначенными задачами,
возможность создавать задачи себе Интеграция с Telegram Запуск внутри Telegram Mini App Авторизация
через Telegram Работа в WebView без отдельной установки приложения 💻 Frontend Стек: React JS

React Router — маршрутизация Context API — глобальное состояние Auth Provider — управление
авторизацией Protected Routes — защита приватных страниц 🧠 Backend Стек: FastAPI

REST API для клиента CRUD‑операции с задачами Управление пользователями и ролями Строгая типизация
данных Валидация входящих запросов Чёткое разделение слоёв логики 🔐 Безопасность и доступ
Авторизация через Telegram Разграничение прав доступа по ролям Защищённые API‑эндпоинты 🚀 Статус
проекта Проект завершён и используется в реальном сценарии
